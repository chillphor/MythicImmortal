# MythicImmortal

---

## English

### 📝 Description
MythicImmortal is a lightweight addon designed for "Mythic Plus Farmers". It automatically tracks the number of Mythic Keystone dungeons completed by all your characters this week (specifically runs with a level of 10 or higher).

### 🚀 How to Use
1.  **Open Interface**: Press `I`  to open the PVE/PVP panel. The addon frame is located on the right side of this panel.
2.  **View Stats**: It automatically displays the completion count for each character and the total weekly runs.
3.  **Send Key**: Click the "Send Key" button to automatically announce your current key level and dungeon in the chat.

### ⌨️ Slash Commands
-   `/mptr` or `/mptr show` : Show the tracking window.
-   `/mptr key` : Manually send the current key info to Party/Raid.
-   `/mptr reset` : **[Caution]** Reset all local data.
-   `/mptr` (alone) : Print a debug report in the chat.

## 简体中文

### 📝 插件介绍
MythicImmortal 是一款专为“大米仙人”设计的轻量级插件。它能自动追踪你本周所有角色的大秘境完成次数（仅统计 10 层及以上的记录），并显示在 PVE 界面侧边。

### 🚀 使用方法
1.  **打开界面**：按 `I`  打开集合石/日历界面，插件框架位于该界面右侧。
2.  **查看统计**：自动显示每个角色的完成次数以及本周总低保次数。
3.  **发送钥匙**：点击 `发送钥匙` 按钮，即可自动将当前钥匙层数和副本发送到队伍/团队频道。

### ⌨️ 命令行
-   `/mptr` 或 `/mptr show` : 显示插件主窗口。
-   `/mptr key` : 手动发送当前钥匙信息到队伍/团队。
-   `/mptr reset` : **[慎用]** 重置所有本地记录数据。
-   `/mptr` (单独输入) : 在聊天框打印调试报告。
